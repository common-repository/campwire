The plugin submits Campwire customer name and email to ActiveCampaign list from purchase Thank You page.

== Description ==
Integration for Campwire plugin submits customer name and email from Thank You page to ActiveCampaign list.
This in turn allows utilization of advanced ActiveCampaign triggers, automation and segmentation.

* Native integration with selected ActiveCampaign list.
* ActiveCampaign tracking code insertion.
* Allows more reliable ActiveCampaign triggers
* Simple to install and use, just insert API keys, select list, and insert shortcode on thank you page
* Helps you to save valuable time through automation.

[GRIT Online](https://www.grit.online/campwire-plugin/ "GRIT Online")

== Installation ==
1. Visit [https://wordpress.org/plugins/](https://wordpress.org/plugins/ "https://wordpress.org/plugins/") and search for "Campwire" by GRIT Online Inc.
1. Download and Install the plugin.
1. Activate the plugin.
1. Click on "settings" and click on "Campwire"
1. Enter your details to customize the plugin as required, and save changes.
1. Configure your Campwire purchase completion settings to redirect to your wordpress website. 
1. Insert the shortcode on the 'thank you' page residing on your wordpress website.
1. Publish the page.

The plugin will collect the Campwire purchase transaction details and inserts the customer details to the selected ActiveCampaign list.
